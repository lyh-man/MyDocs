# 微信小程序入门

## 一、基本概念
### 1、简介
```text
Mini Program，直译：小程序。
小程序 提供了一个简单、高效的应用开发框架和丰富的组件及API。
小程序是 占用极小内存的应用，无需安装占用大内存的 APP，即可体验 APP 的服务。
```
